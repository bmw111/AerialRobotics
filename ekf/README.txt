If you want to run the code - 
1. Make sure you copy the mat files of pertaning trajectories copied in the current folder(DataStraightLine.mat is present currently in this repository)
2. Run ekf.m while setting current directory as current folder. In the first few lines of the script you can change the mat file to the file you want.
3. After Running you should see a scatter plot of the generated points
4. Specific Trajectory plotting is done in the plotting.m function
